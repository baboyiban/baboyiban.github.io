title: 변경사항을 다른 브렌치에서 적용하기
description: 변경사항을 다른 브렌치에서 적용하기
---

에러 메시지에 따르면 여전히 변경되지 않은 파일들이나 추적되지 않은 파일들이 존재합니다. `git stash` 명령어를 실행했지만 아무 변경사항이 없다는 메시지가 나온 것으로 보입니다.

먼저 현재 작업 디렉토리에서 추적되지 않은 파일들을 제거해 보세요. 아래의 명령어를 사용할 수 있습니다:

```bash
git clean -fdX
```

이 명령은 추적되지 않은 파일 및 디렉토리를 제거합니다. `-n` 옵션을 추가하여 제거될 파일들을 먼저 확인할 수 있습니다. 아래와 같이 실행해 보세요:

```bash
git clean -fdXn
```

만약 확인 결과가 원하는 대로라면, `-n` 옵션을 빼고 명령을 실행하여 파일을 실제로 제거합니다:

```bash
git clean -fdX
```

그 후에 다시 브랜치를 전환해 보세요:

```bash
git checkout develop-1.1.1
```

만약 이러한 조치가 여전히 문제를 해결하지 못한다면, 해당 파일들이 다른 프로세스에서 사용되고 있거나 권한 문제가 있는지 확인해보세요.
