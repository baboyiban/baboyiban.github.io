---
title: 변경사항 확인 (git status)
description: 변경사항 확인 (git status)
---

변경된 파일 및 스테이징된 상태를 확인하려면 `git status` 명령어를 사용합니다. 이 명령어는 현재 작업 디렉토리의 상태를 보여줍니다.

```bash
git status
```

`git status`는 다음과 같은 정보를 제공합니다:

- 어떤 파일이 수정되었는지
- 어떤 파일이 스테이징 영역에 있는지
- 어떤 파일이 커밋되기를 기다리고 있는지
- 현재 어떤 브랜치에 있는지

커맨드를 실행하면 변경된 파일, 스테이징된 파일, 커밋되기를 기다리고 있는 파일 등이 목록으로 출력됩니다.
