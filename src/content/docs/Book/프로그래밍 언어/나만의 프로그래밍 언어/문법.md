### 변수
```
// int, str,

n(0): int; // int 생략 가능
string("Hello, World!"): str; // str 생략 가능

value(); // value변수의 값을 반환한다. 10을 반환.
value; // () 생략 가능
```

### 함수
```
// 함수 선언
test(x: int, y: int): int {
    return x + y;
}

test(10, 20) // 30
test(x(10), y(20)) // 30. 명시적으로 특정 변수에 넣기 가능

test2(x: int, y: int): x+y; // 함수 선언
test2(10 + 12); // 24
```
